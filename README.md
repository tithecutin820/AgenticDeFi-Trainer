## ❓ Frequently Asked Questions

**Does the software trade automatically while the computer is off?**
The software requires your computer to be on and connected to the internet to perform active trades. If you exit the application, the agents stop working until you restart the program.

**Do I need to pay for network fees?**
Yes. Every trade on a blockchain requires a fee. The agents account for these fees when calculating potential gains.

**How do I update the software?**
When a new version becomes available, the application notifies you during startup. You can visit the download page again to get the latest version. Installing the new version overwrites the old one but keeps your settings intact.

**What happens if I lose my wallet backup?**
The software stores your recovery code locally. Keep this code in a secure, physical location. If you lose your computer and your backup code, you cannot recover your wallet.
